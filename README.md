# Low Level Design (LLD) - Mastery Path

Structured, topic-wise deep dive into low level design.

## Structure

Each major topic gets its own folder. Subtopics are broken into individual files.
Theory, patterns, trade-offs, and implementation — all covered.

## Topics

1. `01-solid-principles/` - SOLID principles
2. `02-design-patterns/` - Gang of Four + modern patterns
3. `03-oop-concepts/` - OOP fundamentals done right
4. `04-uml-diagrams/` - Class, sequence, activity diagrams
5. `05-case-studies/` - Real system designs (parking lot, elevator, etc.)
6. `06-concurrency/` - Threading, locks, concurrent design
7. `07-api-design/` - REST, contracts, versioning
8. `08-database-design/` - Schema design, normalization, indexing
9. `09-code-quality/` - Clean code, refactoring, testing
